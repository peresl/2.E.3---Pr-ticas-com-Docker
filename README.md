# Docker Fundamentals

Repositório com a solução das atividades práticas propostas no curso do IFPB sobre Docker.

## Estrutura do Projeto

- `01-primeiro-container/`: Script para iniciar e parar um container Nginx.
- `02-container-logs/`: Persistência de logs do Nginx usando volumes.
- `03-blog/`: Deploy do WordPress com MySQL usando Docker Compose.
- `04-ambiente/`: Ambiente com PostgreSQL e pgAdmin para desenvolvimento local.

## Como Executar

```bash
docker-compose up -d
